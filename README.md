NovaTech Sales & Service Dashboard

Overview

The **NovaTech Dashboard** is a web-based business intelligence application designed to analyze smartphone sales, service performance, and customer satisfaction.
This project integrates interactive web technologies with an embedded IBM Cognos Analytics dashboard to provide real-time insights for decision-making.

Objectives

* Monitor overall sales performance
* Analyze return rates and identify problem models
* Evaluate service center efficiency
* Track customer satisfaction trends
* Provide actionable business insights

Technologies Used

* **HTML5** – Structure of the application
* **CSS3** – Styling and responsive layout
* **JavaScript** – Interactivity and logic
* **Chart.js** – Data visualization (Bar, Line, Pie charts)
* **IBM Cognos Analytics** – Embedded dashboard for advanced analytics

Features

* Login system (frontend-based authentication)
* Interactive charts:
  * Sales by Model (Bar Chart)
  * Monthly Sales Trend (Line Chart)
  * Sales Channel Split (Pie Chart)
* Embedded Cognos Dashboard
* Professional UI with cards and grid layout
* Navigation (Login → Dashboard → Logout)
